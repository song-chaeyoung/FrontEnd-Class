Common / Reset CSS

효율적인 코드 작성을 위한 공통으로 Common / Reset CSS 파일을 만들어 작성하였습니다. 기존 코드 외에도 외부에서 가져온 서체등도 함께 관리하여 효율적인 코드 작성을 하고자 하였습니다.

CSS-2

Nesting

CSS Nesting (CSS 중첩)을 활용하여 간결한 코드 작성을 하였습니다. 코드의 전체 길이가 짧아지고, 처음 작성할때는 직관적으로 코드 작성이 가능하였으나, CSS Nesting에 아직 익숙하지 않아 코드 수정 시 부모와 자식 요소가 헷갈리는 문제가 발생하였고 반응형 제작시 자식요소를 수정하려면 네스팅된 요소를 모두 작성해야하여 데스크탑 코드 작성에 비해 비효율적인 부분이 느껴졌습니다.
